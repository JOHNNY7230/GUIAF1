# Guia Fórmula 1 🏎️

Um aplicativo de bolso desenvolvido em Flutter para quem respira Fórmula 1 e quer acompanhar a temporada atual de perto.

## O Que o App Faz
* **Grid de Largada:** Uma lista mostrando os pilotos em atividade e as cores de suas equipes (incluindo as novidades do ano, como Hamilton na Ferrari e Antonelli na Mercedes).
* **Classificação Ao Vivo:** Consome dados diretamente de uma API para mostrar a tabela real e atualizada do campeonato mundial.
* **Menu do Paddock:** Um espaço para tirar dúvidas sobre o regulamento e o acerto dos carros.
* **Favoritos:** Área dedicada para salvar os seus pilotos do coração.

## Tecnologias
* **Dart / Flutter:** Para uma interface rápida, bonita e multiplataforma (Android e iOS).
* **HTTP:** Conexão com a API pública Jolpi F1 (espelho do Ergast) para buscar os dados em tempo real no formato JSON.
